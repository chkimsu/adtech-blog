Thompson Sampling Simulator

Visualize how Thompson Sampling uses Probability Distributions (Beta) to balance exploration and exploitation.

X-Axis (CTR): The probability that a user clicks.

Y-Axis (Density): How confident the model is. Taller/Narrower = More Confident.

How it works

This demo uses Bernoulli Thompson Sampling with Beta distributions.

  • Beta Distribution ($\alpha, \beta$): Represents the probability of success (CTR).
  • $\alpha$ (Alpha): Number of Clicks (Successes) + 1.
  • $\beta$ (Beta): Number of Non-Clicks (Failures) + 1.

Interpretation

  • Shift Right: When you click (üëç), $\alpha$ increases, moving the curve to the right (higher estimated CTR).
  • Shift Left: When you ignore (üëé), $\beta$ increases, moving the curve to the left.
  • Get Narrower: As you provide more feedback (clicks or ignores), the curve becomes taller and narrower. This means the model is becoming more certain about the true CTR.
