예측 모델이 1% 틀리면 입찰가가 1% 틀리고, 매출은 3~5% 흔들린다. 입력단 오차가 비대칭적으로 증폭되는 이유와, 과대입찰이 과소입찰보다 더 큰 손해인 까닭을 슬라이더로 직접 만져보세요.
이 데모가 보여주는 것pCTR 모델의 작은 편향이 입찰가 → 낙찰률 → Surplus(남는 이익)로 연쇄 증폭되는 과정을 슬라이더 하나로 실시간 계산해 봅니다.
왜 알아야 하나"모델 1% 개선"이 매출에서는 3~5%로 증폭됩니다. 특히 과대입찰이 과소입찰보다 더 아픈 비대칭을 알아야 모델 오차의 비용을 제대로 읽을 수 있어요.
    이렇게 해보세요 — 6분
  1. 편향 = 0(기본) 상태에서 계산표 한 번 읽기 → Surplus(이익) 기본값 메모.
  2. 편향을 +10%로 올려보세요 → 과대입찰 시 Surplus가 얼마나 떨어지는지 확인.
  3. 편향을 −10%로 내려보세요 → 과소입찰의 손실은 더 작은 게 보임 (비대칭).
시나리오 — 광고 지면 하나, 진짜 세계의 True 파라미터는 고정입니다: True pCTR = 2.0%, pCVR = 5.0%, Conversion Value = $20 · 따라서 True Value = $0.020 per impression.

DSP의 pCTR 모델이 얼마나 정확한가에 따라 예측 True Value가 달라지고, 그 값을 근거로 입찰가가 결정됩니다. 경매 시장은 경쟁자 입찰가 분포(Log-normal)로 단순화했습니다. "모델 오차가 수익으로 어떻게 번역되는가"를 슬라이더로 직접 만져 확인하세요.

편향별 Surplus 곡선

x축: pCTR 편향(%), y축: 1000 impression당 Surplus. 0% 편향에서 최적점이 형성되며, 양쪽으로 갈수록 과대·과소입찰 손실이 비대칭적으로 커집니다.

이 데모에서 알게 된 3가지

  1. pCTR 모델 정확도가 곧 수익 — 10% 편향이 입찰가 10%를 흔들고, 결국 Surplus를 훨씬 더 크게 흔든다. "모델 개선 1%가 ROAS 3~5%"로 증폭되는 이유.
  2. 과대입찰의 독성 — 낙찰률은 올라도 True Value보다 많이 내면 매 경매가 손실. 편향 축의 양쪽이 비대칭인 이유다.
  3. Calibration의 중요성 — AUC가 높아도 출력 확률이 실제와 다르면(=편향) 이 곡선에서 바로 0% 지점에서 벗어난다. 모델 평가에 Calibration 지표가 필수.

현업에서는 이렇게 씁니다

DSP에서 pCTR 모델 개선은 "오프라인 지표 몇 점"이 아니라 돈의 언어로 번역해 보고합니다. 이 데모의 Surplus 곡선이 바로 그 번역기예요.

  • 모델 투자 설득: "AUC 0.01 개선"보다 "편향 10%p 감소 → 1000 노출당 Surplus $X 회복"이 예산 결정을 움직입니다. 모델 팀과 비즈니스 팀이 같은 곡선을 보고 대화해요.
  • 과대예측 안전장치: 과대입찰 쪽 손실이 더 크기 때문에, 새 모델·새 지면처럼 예측이 불안한 구간에는 bid cap이나 더 보수적인 shading을 걸어 하방을 막습니다.
  • 성수기 재점검: 시장 경쟁도가 오르면 같은 편향이라도 손실 금액이 커집니다. 블랙프라이데이 같은 성수기 전에 모델 보정과 shading 파라미터를 다시 맞추는 이유예요.
