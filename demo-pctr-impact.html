pCTR Model Quality Impact

pCTR 예측 모델의 오차가 입찰가·낙찰률·수익에 어떻게 연쇄적으로 영향을 주는지, 슬라이더 하나로 실시간 계산해 봅니다.

시나리오 — 광고 지면 하나, 진짜 세계의 True 파라미터는 고정입니다: True pCTR = 2.0%, pCVR = 5.0%, Conversion Value = $20 · 따라서 True Value = $0.020 per impression.

DSP의 pCTR 모델이 얼마나 정확한가에 따라 예측 True Value가 달라지고, 그 값을 근거로 입찰가가 결정됩니다. 경매 시장은 경쟁자 입찰가 분포(Log-normal)로 단순화했습니다. "모델 오차가 수익으로 어떻게 번역되는가"를 슬라이더로 직접 만져 확인하세요.

x축: pCTR 편향(%), y축: 1000 impression당 Surplus. 0% 편향에서 최적점이 형성되며, 양쪽으로 갈수록 과대·과소입찰 손실이 비대칭적으로 커집니다.

이 데모에서 알게 된 3가지

  1. pCTR 모델 정확도가 곧 수익 — 10% 편향이 입찰가 10%를 흔들고, 결국 Surplus를 훨씬 더 크게 흔든다. "모델 개선 1%가 ROAS 3~5%"로 증폭되는 이유.
  2. 과대입찰의 독성 — 낙찰률은 올라도 True Value보다 많이 내면 매 경매가 손실. 편향 축의 양쪽이 비대칭인 이유다.
  3. Calibration의 중요성 — AUC가 높아도 출력 확률이 실제와 다르면(=편향) 이 곡선에서 바로 0% 지점에서 벗어난다. 모델 평가에 Calibration 지표가 필수.
