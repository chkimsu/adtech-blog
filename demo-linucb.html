How it works

This demo implements the LinUCB (Linear Upper Confidence Bound) algorithm entirely in your browser.

  • Context ($x$): Each ad has a fixed feature vector (e.g., [Tech Score, Fashion Score, etc.]).
  • Model ($\theta$): The algorithm maintains a belief vector for each ad.
  • Selection: It calculates a score $s = x^T\theta + \alpha\sqrt{x^T A^{-1} x}$.
  • Update: When you click, it updates $A$ and $b$ to refine its prediction for next time.
